## Gaps Found

### GAP 1 — PROCESS_STAGES Name Mismatches (DISP-03 partial fail)

The equipment accordion's stage grouping is broken for most items because `PROCESS_STAGES` in `config.py` was written with assumed component names that differ from the actual xlsx BOM column B strings. Items with no matching stage entry fall into "Other".

#### Hybrid System (12/14 items → "Other")

Only 2 of 14 hybrid items correctly map to a stage:

| xlsx Name | Gets Stage |
|-----------|-----------|
| `1 MW Aeromotor Turbine` | Power & Drive ✓ |
| `Vertical Turbine Pump (PSI Prolew Flowserve VTP)` | Water Extraction ✓ |
| `Gearbox (Winergy  PEAB series) - Must be ordered with second output shaft` | **Other** ✗ |
| `Variable-Displacement Hydraulic Power Unit (HPU)` | **Other** ✗ |
| `Hydraulic Motor (225 kW rating) (Haaglund CA 70)` | **Other** ✗ |
| `Wind turbine rotor lock` | **Other** ✗ |
| `Extra storage tank (100,000 gallons)` | **Other** ✗ |
| `Pipes (total)` | **Other** ✗ |
| `PLC (Siemens SIMATIC S7-1200 CPU1215C-1)` (with `\xa0`) | **Other** ✗ |
| `Battery (Tesla Megapack 3.9MWh unit)` | **Other** ✗ |
| `Pure Aqua Large Reverse Osmosis System RO-600 (...)` | **Other** ✗ |
| `High Pressure Pump (Danfoss APP 78/1500 180B7808 (1300 L/min)` | **Other** ✗ |
| `HCI544E 3-Phase - Stamford \| 600 kW ` | **Other** ✗ |
| `Rectifier/Inverter Package` | **Other** ✗ |

Config hybrid section has stale/wrong names (e.g., `"Gearbox (Winergy PEAB series)"` instead of `"Gearbox (Winergy  PEAB series) - Must be ordered with second output shaft"`, `"Battery (Tesla Megapack 3.9 MWh)"` instead of `"Battery (Tesla Megapack 3.9MWh unit)"`, etc.).

Config also entirely missing 3 hybrid items: `Wind turbine rotor lock`, `HCI544E 3-Phase - Stamford | 600 kW`, `Rectifier/Inverter Package`.

#### Mechanical System (1/14 items → "Other")

| xlsx Name | Gets Stage |
|-----------|-----------|
| `Pure Aqua Large Reverse Osmosis System RO-600 (Includes Pre and Post treatment)` | **Other** ✗ |
| *(all other 13 items)* | correct stage ✓ |

Config has `"Reverse osmosis train"` and `"Calcite bed contactors"` which don't exist in xlsx.

#### Electrical System (1/8 items → "Other")

| xlsx Name | Gets Stage |
|-----------|-----------|
| `Pure Aqua Large Reverse Osmosis System RO-600 (Includes Pre and Post treatment)` | **Other** ✗ |
| *(all other 7 items)* | correct stage ✓ |

**Root cause:** The xlsx consolidates pre-treatment, RO, and post-treatment into a single line item (`Pure Aqua Large Reverse Osmosis System RO-600`) for mechanical and electrical. The config split these into 3 separate phantom entries. The hybrid PROCESS_STAGES was rewritten in Phase 16 with names that were never validated against the actual xlsx.
